# Axon

Visual, node-based canvas for composing multi-step AI agents.

> **Status:** M1 — scaffold + canvas stub. Live: [axon.raymnz.com](https://axon.raymnz.com) (once DNS propagates).

## Stack
Next 16 (App Router) · React 19 · Tailwind 4 · `@xyflow/react` · `framer-motion` · Cloudflare Workers via `@opennextjs/cloudflare` · Gemini 2.5.

## Develop
```bash
pnpm install
pnpm dev
```

## Deploy
```bash
pnpm deploy   # opennextjs-cloudflare build && deploy
```

See `BRIEF.md` for the full product spec and milestone plan.
